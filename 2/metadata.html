'사진'의 메타 데이터 : 찍은 장소, 시간 데이터 등
'오디오파일'의 메타 데이터 : 재생 시간, 채널 수 등

HTML 페이지에 대한 메타 데이터를 담기 위한 태그들 : <base>, <link>, <script>, <style>, <title>, <meta>
메타태그들은 주로 head 태그 안에 작성

웹 페이지의 베이스 URL <base>

웹 페이지들이 담겨있는 기본 URL과 웹 페이지가 출력될 윈도우를 지정하기 위해 사용

메타 데이터 <meta>

웹 페이지의 저작사, 문자 인코딩 방식, 문서 내용 등 다양한 메타 데이터를 표현하기 위해 사용
name과 content의 속성 쌍으로 구성
